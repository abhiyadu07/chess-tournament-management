# Chess Tournament Management System

A simple web application for managing chess players, tournaments, matches, and final rankings.
## Features
- Add and manage chess players
- Create and delete tournaments
- Add players to tournaments
- Create random matches
- Select match winners
- Display final player rankings
- Store tournament data using LocalStorage
## Tech Stack
- Svelte
- JavaScript
- HTML
- CSS
- Vite
